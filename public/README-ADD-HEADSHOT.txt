ADD THE PORTRAIT HERE
=====================

Save Dr. Bharath Kumar's headshot in this /public folder with the
exact filename:

    headshot.jpg

Recommended: a portrait-orientation image, at least 1000 x 1250 px,
with the face in the upper portion of the frame (the hero crops to a
4:5 ratio and anchors to the top). The supplied studio headshot is ideal.

Until the file is added, the hero shows a refined "GBK" monogram
placeholder automatically — nothing breaks. Drop the file in and refresh.

To change the filename or path, edit `profile.headshot` in /lib/data.ts.
